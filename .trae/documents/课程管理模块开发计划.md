## 二、开发任务分解

### 任务1: 课程管理主页面开发

**目标**: 开发课程管理列表页面，与课程管理.png保持一致

**页面结构**:

1. 面包屑导航: 基础信息管理 / 课程管理
2. 主内容卡片(main-card):

   * 页面标题: 课程管理

   * 快捷筛选标签:

     * 左侧：总计(48)、启用(24)、停用(12)

     * 右侧：新增课程按钮

   * 操作栏:

     * 左侧: 搜索框(支持课程名称、学科、年级搜索) + 筛选title和图标，学科及学段下拉框

     * 学科(全部/数学/语文/英语/物理/化学等)

     * 学段(全部/小学/初中/高中等)

   * 数据表格(el-table):

     * 课程名称、课程代码、所属学科、学段、课时名称、课程简介、状态、操作

     * 操作列: 编辑、删除图标

   * 分页组件

**数据字段设计**:

```javascript
{
  id: 1,
  courseName: '语文',
  courseCode: 'CHN001',
  subject: '语文',
  gradeLevel: '小学',
  classHourName: '语文课时',
  description: '语文基础课程',
  status: '启用', // 启用/禁用
}
```

### 任务2: 新增/编辑课程弹窗开发

**目标**: 开发步骤条形式的课程新增/编辑弹窗，与参考图片一致

**弹窗组件**: `CourseDialog.vue`

**弹窗规格**:

* 宽度: 700px

* 标题: 新增课程 / 编辑课程

**步骤设计**(根据用户交互说明):

**步骤1: 基本信息** (对应图片新增课程1)

* 课程名称(必填)

* 所属学科: 下拉选择(语文/数学/英语/物理/化学/生物/历史/地理/政治/体育/美术/音乐/信息技术等)

* 学段：下拉框(小学、初中、高中)

* **重点交互**: 学段选择

  * 小学

  * 初中

  * 高中

  * **说明**: 选择学段后，下一步显示对应课时时长表

**步骤2: 课时安排** (对应图片新增课程2 - 选择学段后)

* 显示根据所选学段对应的课时时长表

* 表格内容:

  * 节次(第1-8节)

  * 课时时长(分钟)

* **说明**: 不同学段对应不同的课时标准时长表

**步骤3: 课时选择配置** (对应图片新增课程3和新增课程4)

* 选择具体课时后显示的配置页面

* 内容:

  * 课时分配

  * 每周上课次数

  * 上课时间偏好

  * 或其他课时相关配置

**交互说明**:

1. 点击"新增课程" → 显示步骤1(基本信息)
2. 在步骤1中选择学段(小学/初中/高中) → 点击下一步
3. 进入步骤2，显示对应学段的课时时长表 → 点击下一步
4. 进入步骤3，进行课时选择配置 → 完成提交

**弹窗规格**:

* 宽度: 800px

* 标题: 新增课程 / 编辑课程
